# ikinciel.dev Clone

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### json-server for local db

```
npm i json-server
```

```
json-server --watch db.json
```
